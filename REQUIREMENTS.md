# MC Helper — 需求文档

**版本**: v0.3.0  
**最后更新**: 2026-03-29  
**状态**: 全部功能已实现（P0 + P1 + P2）

---

## 一、项目背景

原版 Minecraft 缺少很多实用辅助信息，需要频繁打开 F3 调试界面才能查看坐标、光照、方块信息等。本项目旨在提供一个**轻量级、低干扰、开箱即用**的客户端辅助模组，将常用信息以精致的 HUD 形式呈现，提升游戏体验。

---

## 二、目标用户

- 个人自用，Java 版 Minecraft 玩家
- 需要频繁查看坐标/光照/方块信息的玩家（探索、挖矿、建筑、防刷怪）

---

## 三、技术约束

| 项目 | 规格 |
|------|------|
| 游戏版本 | Minecraft Java Edition 1.20.1 |
| 模组加载器 | Forge 1.20.1-47.4.18 |
| 开发语言 | Java 17 |
| 包名 | com.ksw.mchelper |
| 运行端 | 纯客户端（clientSideOnly = true） |
| 性能目标 | 开启全部功能后 FPS 降低 ≤ 5% |

## 四、功能需求

### P0 — 核心功能（已实现 ✅）

#### 4.1 坐标 HUD（快捷键 H）
- 显示玩家精确坐标（XYZ，保留 1 位小数）
- 显示方块坐标（整数坐标）
- 显示朝向（中文方向名 + 角度，精确到 0.1°）
- 显示当前生物群系（中文名称，内置 60+ 种）
- 显示所在维度（主世界 / 下界 / 末地）
- 显示游戏时间（HH:MM 格式 + 白天/夜晚标记）
- 显示天气状态（☀晴天 / ☂雨天 / ⚡雷暴）
- 显示 FPS（绿色≥60 / 黄色≥30 / 红色<30）
- 显示内存使用量（已用/最大 MB，颜色分级）
- F3 调试界面开启时自动隐藏

#### 4.2 方块/实体信息（快捷键 J）
- 准星指向时自动显示
- **方块**：名称、ID、坐标、方块光照（颜色标记）、硬度、推荐工具
- **实体**：名称、ID、分类（敌对/友好/玩家/中立）、血量血条、护甲值

#### 4.3 光照等级叠加层（快捷键 O）
- 红/黄/绿三色标记刷怪危险等级
- 渲染范围最大 64 格，每 10 tick 更新，圆形扫描

### P1 — 扩展功能（已实现 ✅）

#### 4.4 装备耐久 HUD（快捷键 K）
- 主手/副手/头盔/胸甲/护腿/靴子耐久实时监控
- 颜色分级 + 低耐久闪烁警告（< 15%）
- 右下角显示，自动隐藏

#### 4.5 迷你地图（快捷键 M）
- 右上角俯视地图，MapColor 着色
- 玩家实时位置（无滞后）+ 朝向箭头
- 高度明暗立体感
- **异步扫描 + NativeImage 纹理**，FPS 影响 < 1%

#### 4.6 刷怪检测（快捷键 N）
- 扫描周围 32 格内的敌对生物，按类型统计
- 危险等级颜色提示（绿/黄/红）

#### 4.7 合成查询（快捷键 R）✅
- 手持物品时显示合成配方
- 支持有序合成（3x3 / 2x2）、无序合成、烧炼（熔炉/高炉/烟熏炉）
- 多配方时显示第一种并标注总数量
- 显示位置：屏幕居中下方

### P2 — 高级功能（已实现 ✅）

#### 4.8 建筑辅助（快捷键 B）
- **地面网格线**：玩家脚下 8 格范围蓝白色方块网格
- **水平高度线**：玩家当前 Y 高度的青色水平参考线（12 格范围）
- **准心方块高亮**：黄色半透明覆盖目标方块顶面
- **HUD 信息**：当前 Y 高度 + 准心方块坐标（右下角）

## 五、非功能需求

| 需求 | 描述 |
|------|------|
| 性能 | 开启全部功能 FPS 降低 ≤ 5% |
| 兼容性 | 不修改游戏文件，纯事件驱动 |
| 配置 | Forge 配置系统，TOML 格式，客户端独立 |
| 本地化 | 内置中文（zh_cn）和英文（en_us）翻译 |
| 可关闭 | 每个功能可独立通过快捷键开关 |
| 轻量 | 不依赖第三方库，只使用 Forge MDK 提供的 API |

---

## 六、快捷键说明

| 快捷键 | 功能 | 默认状态 |
|--------|------|----------|
| H | 坐标 HUD | 开启 |
| J | 方块/实体信息 | 开启 |
| O | 光照叠加层 | 关闭 |
| K | 装备耐久 HUD | 开启 |
| M | 迷你地图 | 关闭 |
| N | 刷怪检测 | 关闭 |
| R | 合成查询 | 关闭 |
| B | 建筑辅助 | 关闭 |

> 所有快捷键均可在游戏设置 → 控制 → 按键设置 中自定义
