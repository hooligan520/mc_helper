# MC Helper — 需求文档

**版本**: v0.2.0  
**最后更新**: 2026-03-29  
**状态**: P0 + P1（部分）+ P2（部分）已实现

---

## 一、项目背景

原版 Minecraft 缺少很多实用辅助信息，需要频繁打开 F3 调试界面才能查看坐标、光照、方块信息等。本项目旨在提供一个**轻量级、低干扰、开箱即用**的客户端辅助模组，将常用信息以精致的 HUD 形式呈现，提升游戏体验。

---

## 二、目标用户

- 个人自用，Java 版 Minecraft 玩家
- 需要频繁查看坐标/光照/方块信息的玩家（探索、挖矿、建筑、防刷怪）

---

## 三、技术约束

| 项目 | 规格 |
|------|------|
| 游戏版本 | Minecraft Java Edition 1.20.1 |
| 模组加载器 | Forge 1.20.1-47.4.18 |
| 开发语言 | Java 17 |

| 运行端 | 纯客户端（clientSideOnly = true） |
| 性能目标 | 开启全部功能后 FPS 降低 ≤ 5% |

## 四、功能需求

### P0 — 核心功能（已实现）

#### 4.1 坐标 HUD（快捷键 H）
- 显示玩家精确坐标（XYZ，保留 1 位小数）
- 显示方块坐标（整数坐标，建筑/挖矿定位用）
- 显示朝向（中文方向名 + 角度，精确到 0.1°）
- 显示当前生物群系（中文名称，内置 60+ 种）
- 显示所在维度（主世界 / 下界 / 末地）
- 显示游戏时间（HH:MM 格式 + 白天/夜晚标记）
- 显示天气状态（☀晴天 / ☂雨天 / ⚡雷暴）
- 显示 FPS（绿色≥60 / 黄色≥30 / 红色<30）
- 显示内存使用量（已用/最大 MB，颜色分级）
- F3 调试界面开启时自动隐藏（避免信息重叠）
- 支持自定义位置（配置文件 posX/posY）

#### 4.2 方块/实体信息（快捷键 J）
- 准星指向时自动显示，无需按住按键
- **方块信息**：
  - 方块中文名称 + 命名空间 ID（如 `minecraft:grass_block`）
  - 方块坐标
  - 方块光照等级（0=红色危险，1~7=黄色警告，≥8=绿色安全）
  - 破坏硬度值（不可破坏 / 数值）
  - 推荐工具（镐/斧/锹/锄/手撸/任意）
- **实体信息**：
  - 实体名称 + 命名空间 ID
  - 实体分类（敌对/友好/玩家/中立，带颜色区分）
  - 当前血量 / 最大血量 + 血条可视化
  - 护甲值（有护甲时显示）

#### 4.3 光照等级叠加层（快捷键 O）
- 在方块表面绘制半透明颜色叠加层
- 颜色含义：
  - 🔴 红色：方块光照 = 0（必定刷怪）
  - 🟡 黄色：方块光照 1~7（夜晚可能刷怪）
  - 🟢 绿色：方块光照 ≥ 8（安全）
- 渲染范围：玩家周围 16 格（可配置，最大 64 格）
- 只扫描可站立表面（固体方块顶面且上方两格为空气）
- 性能优化：每 10 tick 扫描一次，圆形范围，开启立即扫描

### P1 — 扩展功能（已实现）

#### 4.4 装备耐久 HUD（快捷键 K）
- 显示装备部位：主手、副手、头盔、胸甲、护腿、靴子
- 每个装备显示：物品图标 + 耐久数值（当前/最大）+ 耐久条
- 耐久条颜色分级：绿色（>50%）/ 黄色（>25%）/ 红色（≤25%）
- 低耐久警告：< 15% 时文字闪烁
- 只显示有耐久度的物品（工具、武器、护甲）
- 位置：右下角，避开快捷栏
- 未装备任何耐久物品时自动隐藏

#### 4.5 迷你地图（快捷键 M）
- 右上角显示玩家周围区域的俯视地图
- 使用 MapColor 着色（与原版地图颜色一致）
- 玩家实时位置（永远在中心，无滞后）和朝向箭头
- 高度明暗对比增强地形立体感
- 地图下方实时显示当前 X/Z 坐标
- 性能优化：
  - **异步线程扫描**，完全不阻塞主线程
  - **NativeImage + DynamicTexture**，每帧一次 blit 渲染
  - 每 40 tick（2 秒）刷新一次，移动超过 16 格立即刷新

#### 4.6 刷怪检测（快捷键 N）
- 扫描玩家周围指定范围内（默认 32 格）的敌对生物
- 按实体名称统计数量，按数量降序排列显示
- 危险等级颜色提示：绿色（安全）/ 黄色（≥5 只）/ 红色（≥10 只）
- 实时显示合计数量

#### 4.7 合成查询（P1，待开发）
- 查询物品合成配方

### P2 — 待开发功能

## 五、非功能需求

| 需求 | 描述 |
|------|------|
| 性能 | 开启全部功能 FPS 降低 ≤ 5% |
| 兼容性 | 不修改游戏文件，纯事件驱动 |
| 配置 | Forge 配置系统，TOML 格式，客户端独立 |
| 本地化 | 内置中文（zh_cn）和英文（en_us）翻译 |
| 可关闭 | 每个功能可独立通过快捷键开关 |
| 轻量 | 不依赖第三方库，只使用 Forge MDK 提供的 API |

## 六、快捷键说明

| 快捷键 | 功能 | 默认状态 |
|--------|------|----------|
| H | 坐标 HUD 开/关 | 开启 |
| J | 方块/实体信息开/关 | 开启 |
| O | 光照叠加层开/关 | 关闭 |
| K | 装备耐久 HUD 开/关 | 开启 |
| M | 迷你地图开/关 | 关闭 |
| N | 刷怪检测开/关 | 关闭 |

> 所有快捷键均可在游戏设置 → 控制 → 按键设置 中自定义
